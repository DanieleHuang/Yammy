# Yammy
### Requirements:
1. Organize Food items by restaurants serving them
2. Track the cost for each food item, maybe track how many items you ate it
3. For each item add:
   - Notes (opt)
   - Picture
   - Rating
   - Mood (opt)
4. Filter by Rating, Mood, Price

### Technicals:
1. MERN stack
2. Use google api when adding restaurants
